# 타입별 이미지 업로드 가이드

이 폴더에 각 여성 타입별 이미지를 업로드하세요.

## 필요한 이미지 파일

다음 파일명으로 이미지를 준비해주세요:

1. **cheerful.jpg** - 밝고 활발한 타입
   - 밝은 표정, 에너지 넘치는 느낌
   - 추천: 밝은 색상 의상, 웃는 모습

2. **gentle.jpg** - 조용하고 차분한 타입
   - 차분한 표정, 우아한 느낌
   - 추천: 파스텔 톤, 잔잔한 미소

3. **homely.jpg** - 가정적인 타입
   - 따뜻한 느낌, 편안한 분위기
   - 추천: 일상적인 모습, 자연스러운 표정

4. **educated.jpg** - 지적이고 교양있는 타입
   - 지적인 느낌, 세련된 분위기
   - 추천: 깔끔한 의상, 자신감 있는 표정

5. **traditional.jpg** - 전통적 가치관 타입
   - 단정한 모습, 우아한 느낌
   - 추천: 전통적인 느낌, 고전적인 아름다움

6. **beautiful.jpg** - 젊고 아름다운 타입
   - 생동감 있는 느낌, 젊은 분위기
   - 추천: 트렌디한 스타일, 밝은 표정

## 이미지 권장 사양

- **해상도**: 최소 800x600px (16:9 비율 권장)
- **파일 형식**: JPG, PNG
- **파일 크기**: 500KB 이하 (웹 최적화)
- **주의사항**: 개인정보 보호를 위해 얼굴 모자이크 처리 권장

## 업로드 후

이미지를 업로드한 후, 매칭 페이지에서 자동으로 표시됩니다.
플레이스홀더가 실제 이미지로 교체됩니다.
